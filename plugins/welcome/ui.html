Create a graph by using the toolbox on the left panel.

Element selection can toggled by clicking on an element. Deselect all elements by clicking the background.

To create a new edge, drag the blue handles that appear around a node and drag it to another node. Dragging a spikey corner creates a directed edge while dragging a rounded corner creates a undirected edge

To edit an element's label or weight, double-click on it.

You can access additional functionality by choosing a plugin below.
